## Installation

First, alias the `qub` command to the following:

```
alias qub="curl -fsSL https://raw.githubusercontent.com/jamonholmgren/qub/main/src/cli.sh | bash -s --"
```

Now, you should be able to run it like so:

```
qub
qub --version
qub --help
```
