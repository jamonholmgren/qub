## Installation

First, alias the `qub` command to the following:

```
alias qub="source <(curl -sSL https://raw.githubusercontent.com/jamonholmgren/qub/main/src/cli.sh)"
```

Now, you should be able to run it like so:

```
qub
qub --version
qub --help
```
